Installation of the theme

Realize the following steps to upload the theme to your server:

  • After unzip Theme Files, you will found HTML Folder, all this files you will upload.

  • You can view this Theme in all these browsers (Chrome, Opera, Safari 5, Safari 4, Firefox 4, Firefox 3, IE10, IE9, IE8) from your desktop computer. Because the files are written in HTML, you don't need an internet connection to display or edit the Theme.

  • Start your FTP client (I persoanlly use Filezila which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.

  • Locate the domain folder you wish to upload your Theme files to on your server. You are going to want to click and drag every file that you have labeled .html, .css, .js into your FTP client.

Css Files

Sweet Cake uses different css files that you can find it in the css folder.

  • Style.css, is the main style of the theme that defines all the graphics of the various sections. The file is well commented, all rules are divided by section (navigation, slide, services, testimonials, works, prices, team, skills, and social contacts)

  • Grid.css, this style defines all grids of the site for all types of devices. Don't edit this file.

  • Responsive.css, use this css to define the rules for different devices. Don't edit these rules for correct display on all devices.

  • Isotope.css, this css file is used only for works section that use the plug-in isotope filtering. Don't edit the following file, in case you can consult the online documentation of the author here.

  • PrettyPhoto.css, this css file is used only for the section works that use the plug-in prettyphoto. Don't edit the following file, in case consult the online documentation of the author here.

Header

The header of the site consists of the menu item and the logo. For a correct visualization, enter 4 menu items in the right and 4 items in the left menu. To change the menu items is very simple, look at the code and change only the text within the tag "a". Also change the logo is very simple, my advice is not to edit the HTML but simply replace the current logo with your company logo. Be careful to rename your logo "logo.png" and for proper display maintains the same size: 180px x 239 px.
